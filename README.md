# Millennial Realty &amp; Investments LLC

## Greetings!

This project is a real estate portfolio that connects with the Google Drive API to easily manage media assets, listings, and agents. This support will allow the client to add, edit, or remove media content, property listings, or agent contacts on their website through the comfort and simplicity of Google Drive.

## Tools

Frontend: [Next.js](https://nextjs.org), [React](https://reactjs.org), [Material UI](https://mui.com), [Emotion](https://emotion.sh/docs/introduction)

Backend: [Google Drive](https://google.com/drive)

Testing: [Jest](https://jestjs.io)

## How to use

```
npm install
npm run dev
```

## Testing

```
npm test
```
